# 全局规则

- 所有对话和输出必须使用中文
- 创建自定义清单弹窗中的检索条件必须与主数据清单（DataListTab.vue）上方的检索条件保持同步，包括条件字段、选项列表、交互方式等
- 任何涉及删除数据库数据的操作前，必须向用户明确确认，列明即将删除的具体数据（包括但不限于：表名、记录ID、关键字段值），未经用户明确确认不得执行删除操作
- 任何涉及修改数据库数据的操作前（包括但不限于增删改），必须向用户明确确认，列明即将执行的具体操作、涉及的表名、记录ID、关键字段值，除非用户明确指令可以直接操作数据库，否则未经用户确认不得执行
- 每次完成代码修改后，必须进行严格验证（前端构建、后端编译、功能逻辑检查），确认功能真正实现后再结束工作
- 每次重启前后端服务后，必须验证两个服务均正常响应（curl 检查 8080 和 5173 端口返回 200），如失败则重新启动直至成功
- 每次代码修改完成后，必须检查前后端是否能正常运行：前端执行 npm run build 确认无构建错误，后端执行 mvn compile 确认无编译错误，然后重启服务并用 curl 验证端口响应正常，确认页面无 500/400 等接口报错后才可结束
