# 添翼产品清单管理工具 — 设计文档

## 概述

一个基于 Web 的工具系统，用于管理"添翼产品清单"基础数据，支持版本化数据维护、多维查询和文档生成。

## 角色与权限

### 三种角色

| 角色 | 核心权限 |
|------|---------|
| **管理员** | 用户管理、权限套餐管理（预设+精细调整）、版本创建与封板 |
| **普通用户** | 数据工作台（查询+维护），在版本窗口期内可编辑数据（L3~Ln 的增删改） |
| **高级用户** | 数据工作台（查询+维护+文档生成），可生成招标参数/功能说明 → Word/Excel |

### 权限体系

- **粒度**：功能权限（页面/按钮级别）
- **分配方式**：权限套餐预设（默认勾选一批权限），支持逐项精细调整

## 版本管理

### 流程

1. 管理员创建新版本
   - 自动生成版本号（1.0 → 1.1 → 1.2 ...，小数点后第一位递增）
   - 复制上一已发布版本的全量数据作为基线
   - 版本状态变为"编辑中"
2. 普通用户在窗口期内维护数据
3. 管理员封板发布
   - 记录发布人、发布日期
   - 系统对比上一已发布版本，逐行逐字段生成变更记录（新增/修改/删除）
   - 版本状态变为"已发布"

### 变更记录

每条变更记录包含：层级路径、变更类型（新增/修改/删除）、变更字段、旧值、新值、操作人。

## 数据模型

### 统一树形结构

采用单表 `data_entry` + `parent_id` 实现无限层级：

| 层级 | 名称 | 来源 |
|------|------|------|
| L1 | 业务分类 | 固定数据 |
| L2 | 业务域 | 固定数据 |
| L3 | 产品/系统 | 用户维护 |
| L4 | 模块 | 用户维护 |
| L5 | 功能 | 用户维护 |
| L6 | 子功能 | 用户维护 |
| ... | 无限延伸 | 用户维护 |

所有 46 列 Excel 字段存储在 `data_entry` 表中，每行可包含完整的字段数据。

### 主要表结构

- `sys_user` — 用户（username, password, nickname, role_id, status）
- `sys_role` — 权限套餐（name, code, description）
- `sys_role_menu` — 角色-功能权限关联
- `sys_menu` — 功能权限树（页面/按钮）
- `data_version` — 版本（version_no, status, released_at, released_by）
- `data_version_changelog` — 变更记录
- `data_entry` — 统一数据条目（parent_id, level, sort_order, is_leaf, +46个字段列）

## 前端页面结构

### 管理员
- **用户管理** — 增删改用户，分配权限套餐
- **权限套餐管理** — 创建/编辑套餐，勾选功能权限
- **版本管理** — 创建版本、查看版本列表、封板发布、查看变更记录

### 普通用户
- **数据工作台** — 数据查询 + 数据维护（版本编辑窗口期内可编辑）

### 高级用户
- **数据工作台** — 数据查询 + 数据维护（版本编辑窗口期内可编辑）+ 文档生成

## 数据工作台（统一页面）

数据维护与数据查询共用统一页面，根据权限控制功能按钮的显示。

### 流程说明

**版本选择：**
- 页面初始无版本，弹窗展示所有已发布版本列表供选择
- 点击"切换版本"按钮可重新弹窗选择
- 选择后，顶部显示当前版本信息

**层级树：**
- 根据选中版本的数据，左侧展示 L1~L3 的层级树
- 点击节点可筛选右侧数据

**Tab 1 - 统计视图：**
- 产品总数、模块总数、功能总数、子功能总数（数字卡片）
- 各业务域产品分布（饼图）
- 各状态产品分布（柱状图）
- 本版本变更记录数

**Tab 2 - 数据清单：**
- 查询条件 + 结果表格
- 每行右侧有"编辑"和"删除"按钮（版本为"编辑中"且用户有维护权限时显示）
- "新建"按钮和"生成文档"按钮（"生成文档"仅高级用户可见）

## 技术栈

| 层 | 技术 |
|----|------|
| 前端 | Vue3 + Element Plus |
| 后端 | Spring Boot + Spring Security + JWT |
| 数据库 | 开发期 SQLite，保留 PostgreSQL 迁移脚本 |
| 文档生成 | Apache POI (Word/Excel) |
